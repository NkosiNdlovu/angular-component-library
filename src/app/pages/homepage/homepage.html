Angular component Libray

Get started

Install angular material dependencies

npm install --save @angular/material @angular/cdk

Import pre compiled theme

@import "~@angular/material/prebuilt-themes/indigo-pink.css";

Install library

npm install C:\Projects\nvisionit-angular-component-library\dist\nvisionit-angular-component-library-1.0.1.tgz
